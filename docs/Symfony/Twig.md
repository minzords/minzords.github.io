# Afficher une variable
Dans cet exemple, je vais afficher la variable ave.
```twig
	{{ ave }}
```

# Les boucles
## FOR
```twig
	{% for user in users %}
        <li>{{ user.username }}</li>
  {% endfor %}
```

# Les Conditions
## IF
```twig
	{% if variable == truc) %}
    <p>La valeur de variable est truc.</p>
	{% endif %}
```
